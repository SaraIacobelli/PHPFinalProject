public
  index.php --> pagina iniziale che gestisce il progetto
config
  container.php --> é il contenitore delle dipendenze: contiene cioè le dipendenze di tutte le classi del progetto
  route.php --> contiene lo smistamento delle pagine php in base alle GET
src
  Controller --> contiene tutti i file con il cuore della pagina: qui si includerà il file di connessione al database, si eseguono
                  le query e si inserisce il collegamento al layout passando gli eventuali parametri richiesti dalle view.
    home.php
    article.php
    register.php
    admin.php
  View --> contiene tutti i layout del progetto: è codice html con variabili php il cui valore viene passato nei controller
    home_layout.php
    article_layout.php
    register_layout.php
    admin_layout.php
    error_layout.php
  Model
    conn.php --> file di connessione al database
